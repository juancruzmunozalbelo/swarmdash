🏛 SwarmDash

Real-time control panel for autonomous AI agent swarms.

Circuit breakers, Kanban, SLO monitoring — zero dependencies.

⭐ View on GitHub ↧ Download ZIP

Features

🏛

Agent Visualization

Pixel-art map of your agent swarm with real-time status. Each god represents an autonomous agent.

🔄

Circuit Breakers

Visual health indicators (closed/open/half-open) to prevent cascading failures across agents.

📄

Kanban Board

Track tasks across Backlog, WIP, and Done lanes. Sync with your agent runtime.

📊

Runtime Metrics

SLO monitoring, error budgets, and performance counters updated in real time.

🚨

Incident Runbook

Log incidents, track resolution, and export reports with one click.

🎯

Zero Dependencies

Pure HTML, CSS, and JavaScript. No build step, no node_modules. Just open and go.

Get Started

# Clone the repo
git clone https://github.com/juancruzmunozalbelo/swarmdash.git
cd swarmdash

# Serve with any static server
python3 -m http.server 8888

# Open http://localhost:8888
